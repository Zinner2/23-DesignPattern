# 01-单例模式 Sing0leton Pattern
## A.恶汉式
创建步骤:</br>
  1.构造方法私有化</br>
  2.在类的内部创建对象</br>
  3.向外界暴露一个静态的公共方法</br>
缺点:类装载时完成实例化,如果该实例未使用,可能造成内存浪费。
